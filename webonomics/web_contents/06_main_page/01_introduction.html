6.1. Main Page¶

In this section we will create the basic Main Page for our E-Commerce Website - MyZone.in

6.1.1. Requirement¶

Let us have a look on the basic requirement of this page :

Our final page would look like this :

Note: The colors in the above image may vary from actual output.

Web Page should have :

  1. A header with the company logo, Search bar, User name if any user has logged in and Logout link.
  2. A slider banner displaying the images for current offers and promotions going on.
  3. A category section which lists all the category in the Database of MyZone, in a table format.
  4. Each category is linked to the products page of its kind.
  5. A Footer, which lists other information - About Us, Contact Us etc.
